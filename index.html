PLANET INVADERS

Planet Invaders

// Stack

Ruby on Rails Javascript Html Css

// Project

Planet invaders is a one week team project made at Le Wagon. It's a copycat of Airbnb where you can rent planets based on Star Wars Universe. It was my first team project and we basically used everything we learn't during the bootcamp. We designed first a wireframe on figma and then coded the plateform. The backend is on ruby on rails with a standard mvc model and the database is in MySql, using active record for our requests. For the the frontend, we used traditional html, css & javascript. Overall, it was a very interesting project where I learn't to collaborate with others and how to really use versionning tools such as git and github.

WHISP

Whisp

// Stack

Ruby on Rails React.js Next.js Html Css

// Project

Whisp brings together sports enthusiasts on the equipment topic. We help athletes to find the equipment that works best for them, by helping them discover the equipment used by their favorite friends and athletes.
I joinded a team of 3 developpers right after their MVP creation. At the time, the product was build in Ruby on Rails. We then moved to the first version of the product with a backend still in RoR and a frontend in React.js. I was in charge to developp new features from back to front, such as the home page, the profile page & the fidelity program with the rewards. Thanks to a designer, we had a solid design ready to bed coded.
After few months, we realise the issue of react hydration process and its poor performance, especially for the user who had to wait few seconds for its content to load. We also had an SEO issue. That's why we moved to next.js in order to boost the product performances and improve the user experience.
Overall, I really appreciated this experience. I learnt 2 news framework and reinforce my idea to spesialised myself in frontend development, something I really appreciate and looking forward to develop.

https://www.whisp.social/
